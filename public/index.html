If you have the react dev-tools extension enabled and your browser dev-tools open, you will get parasitic renders because the devtools will read data in the reactive state and subscribe components to it.

Click on a button to render the left or right component, click on "Render/Rerender App" to rerender the app.

Component borders flash red when they render. Notice how on page load all components are red but when you change some piece of state, only the component whose rendered DOM depend on that piece of state rerender.

You can take a look at the code on github
